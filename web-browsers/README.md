### [Ladybird](https://github.com/LadybirdBrowser/ladybird) ⭐ 61.4k

**The first truly independent browser engine built from scratch in decades.**

Every other browser runs on Blink (Chrome, Edge, Brave, Opera), Gecko (Firefox), or WebKit (Safari). Ladybird shares zero code with any of them — it's a completely fresh engine built on libraries originating from SerenityOS: LibWeb for rendering, LibJS for JavaScript, LibWasm, LibCrypto, LibHTTP, and more. Multi-process architecture with sandboxed per-tab renderers, out-of-process image decoding and networking, cross-platform (Linux, macOS, Windows via WSL2).

**Stack:** C++ · Rust · JavaScript · CMake · Python
**License:** BSD-2-Clause
**Status:** Pre-alpha — currently for developers only, not for daily use

**Why it's notable:** Browser engine diversity is critical to the health of the web. When one engine dominates (see: Blink), web standards bend to its implementation decisions. Ladybird is the most significant new browser development in decades — a genuine attempt to build an independent engine from nothing. 61.4k stars on pre-alpha developer software is a clear signal the community understands the stakes. Watch this one closely.
