### [AIRI](https://github.com/moeru-ai/airi) ⭐ 35k

**Self-hosted AI virtual companion with voice, Live2D/VRM models, and game-playing abilities.**

Open-source recreation of Neuro-sama — an AI that can hold real-time voice conversations, animate a Live2D or VRM avatar, and actually play games (Minecraft, Factorio, Kerbal Space Program). Runs fully self-hosted across Web, macOS, Windows, and mobile via PWA. Connects to any LLM provider, handles client-side speech recognition and synthesis, and stores memory using DuckDB WASM in-browser — no server required.

**Stack:** TypeScript · Vue.js · Rust · Tauri · WebGPU · WASM · DuckDB WASM · HuggingFace Candle · Capacitor
**License:** MIT
**Platforms:** Web · macOS · Windows · Mobile (PWA)

**Why it's notable:** The engineering ambition here is real — WebGPU, client-side ML via WASM, in-browser databases, and actual game-playing agents, all in one project. 35k stars and 3.5k forks on what is genuinely niche subject matter signals a large latent demand for self-hosted AI companions done properly. The stack choices (Tauri over Electron, WebGPU over cloud inference) reflect the same offline-first philosophy as the best tools in this collection.
