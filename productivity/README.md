### [AppFlowy](https://github.com/AppFlowy-IO/AppFlowy) ⭐ 69.1k

**Self-hosted Notion alternative — AI-powered workspace built with Flutter and Rust.**

AppFlowy is an open-source collaborative workspace: rich document editing, grid databases, kanban boards, calendar views, and team wikis — all in one cross-platform app. Built with Flutter (UI) and Rust (backend/local storage), it ships native desktop apps for macOS, Windows, and Linux, plus mobile apps for iOS and Android. Self-host the sync server for teams, or run locally without any server at all. AI assistant built in, powered by the LLM of your choice.

**Stack:** Flutter, Rust, self-hosted sync server
**License:** AGPL-3.0

**Why it's notable:** Notion became the default collaborative workspace for a generation of teams, then introduced pricing cliffs, opaque AI add-ons, and terms that make data portability uncomfortable. AppFlowy's answer is principled: local-first data ownership, open source, native performance. 69.1k stars and a last commit on April 5, 2026 (today) signals a project at peak momentum. The Flutter + Rust architecture choice is a long-term bet on performance and cross-platform consistency — the codebase is designed to outlast Electron-based competitors.
