### [Vaultwarden](https://github.com/dani-garcia/vaultwarden) ⭐ 57.9k

**Self-hosted Bitwarden server in Rust — full-featured, uses a fraction of the resources.**

Vaultwarden is an unofficial, community-built implementation of the Bitwarden server API, written from scratch in Rust. It's fully compatible with all official Bitwarden clients (browser extensions, desktop apps, iOS, Android) — your family or team connects their existing Bitwarden apps and nothing changes on their end, except the vault lives on your server. Supports personal vaults, organisation password sharing, FIDO2/WebAuthn/YubiKey 2FA, secure file attachments, and an admin panel. Deploy in a single Docker container.

**Stack:** Rust, Docker
**License:** AGPL-3.0

**Why it's notable:** The official Bitwarden server is Java-based and requires ~2GB RAM. Vaultwarden runs the same workload in ~10MB of RAM — that's the Rust rewrite advantage. AGPL-3.0 keeps it honest. But the number that stands out most: **12 open issues for 57.9k stars** — the lowest issue-to-star ratio in this entire collection by a wide margin. It's a sign of a codebase that's been polished to near-perfection, maintained with extraordinary care, and used reliably in production by a huge community.
