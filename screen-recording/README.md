### [Recordly](https://github.com/webadderall/Recordly) ⭐ 2.8k

**Screen recorder with built-in editor and presentation polish — open-source alternative to Screen Studio.**

Records your screen or a specific window, then gives you a timeline editor to add zoom effects, cursor animations, webcam overlays, annotations, and styled backgrounds before exporting to MP4 or GIF. The standout feature is auto-zoom suggestions driven by cursor activity — it watches where you're clicking and proposes zooms automatically, which is what makes paid tools like Screen Studio worth $100+.

**Stack:** Electron · TypeScript · React · PixiJS · Vite · ScreenCaptureKit (macOS) · Windows Graphics Capture
**License:** MIT
**Platforms:** macOS · Windows (Linux: partial support)

**Why it's notable:** The OSS space has plenty of basic screen recorders but nothing with this level of presentation tooling. Cursor motion blur, click bounce, webcam bubble, frame styling with gradients and padding — this is demo/tutorial-quality output from a free, local tool. Actively developed with only 14 open issues, which is a healthy sign this early.
